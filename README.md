# ⚒️ FocusForge

> **An AI-powered productivity companion that transforms focus sessions into accountable productivity through intelligent task verification, adaptive distraction monitoring, voice interaction, and personalized recommendations.**

---

# 📌 Problem & Domain

### Problem

Traditional productivity applications primarily measure **time spent** rather than **actual work completed**. Users can easily complete focus sessions without accomplishing meaningful tasks, leading to false productivity metrics and poor study habits. Existing solutions also lack intelligent accountability, personalized guidance, and adaptive focus enforcement.

# 🎯 Objective

### Target Users

- Students
- Professionals
- Competitive Exam Aspirants
- Remote Workers

### Pain Point

People frequently lose focus due to distractions, procrastination, and multitasking. Existing focus timers cannot verify whether meaningful work was completed or encourage long-term discipline.

### Our Solution

FocusForge combines Artificial Intelligence, gamification, and adaptive focus modes to ensure users remain productive throughout their study sessions. Instead of rewarding users for merely running a timer, the application rewards verified work completion using AI-powered image verification and intelligent productivity tracking.

# 🏆 Sponsored Track

- ✅ Expo Track – Built using Expo
- ✅ Neo4j Track – Uses AuraDB as primary database
- ✅ Sarvam AI Track – Uses Sarvam AI APIs for voice interaction

### How we used Expo

FocusForge was developed entirely using Expo and React Native, enabling rapid cross-platform development while leveraging native device capabilities such as the camera, notifications, and audio services.

### How we used Neo4j

Neo4j AuraDB powers our AI Coach by storing relationships between users, completed tasks, and task categories. This graph-based structure enables intelligent, personalized task recommendations that adapt to each user's productivity history.

### How we used Sarvam AI

Sarvam AI powers both Speech-to-Text and Text-to-Speech features within FocusForge. Users can create focus sessions by speaking their tasks instead of typing, while the application can also read feedback aloud, making the experience faster, more accessible, and user-friendly.

# 🧪 How to Run the Project

## Requirements

- Node.js
- Expo CLI
- Android Studio or Expo Go
- Firebase Project
- Neo4j Aura Instance
- Google Gemini API Key
- Sarvam AI API Key

---

## Environment Variables

Create a `.env` file inside the backend folder.

```env
GEMINI_API_KEY=

SARVAM_API_KEY=

NEO4J_URI=

NEO4J_USERNAME=

NEO4J_PASSWORD=
```

---

## Local Setup

### Frontend

```bash
git clone https://github.com/evapatel-22/FocusForge.git

cd FocusForge

npm install

npm start
```

### Backend

```bash
cd backend

npm install

node server.js
```
